## 🚀 Current Projects

### [Multi AI (ADK)](https://github.com/javalabgit/adk)
A **modular multi-agent platform** built on Google's Agent Development Kit. Features specialized agents for travel planning, data analysis, medical consultation, and IT services—with a meta-agent that dynamically creates custom agents on demand.

*Why it matters*: Demonstrates clean agent composition, domain specialization, and scalable architecture for task-specific AI.

### [NewsOS](https://github.com/javalabgit/My-news)  
**Personal intelligence engine** that aggregates global news and analyzes it through your life context. Built with FastAPI + React, powered by CrewAI agents running Gemini analysis.

*Key features*:
- Real-time RSS aggregation from 15+ sources
- Dual-agent intelligence pipeline (curator + strategist)
- Personalized risk/opportunity detection
- Contextual JSON outputs for structured dashboards

### [HAI Render](https://github.com/javalabgit/hai-render)
**AI-powered educational platform** with document chat, video integration, Q&A threads, and auto-generated course materials. Combines modern ed-tech with LLM document understanding.

*What I shipped*: Modular components (document parsing, transcript extraction, quiz generation), real-time student dashboards, assignment workflows.
